# ANAI Athleisure Website Plan 02: Design, Layout, and Product Direction

## Goal

Design ANAI as a clean athleisure ecommerce website with an editorial shopping feel. The site should use strong lifestyle images, clear product rows, category blocks, activity-based shopping, and outfit-led selling.

## Brand colours

| Colour | Hex | Use |
|---|---|---|
| Deep Plum | #461828 | Footer, premium blocks, campaign panels, small labels |
| Burnt Clay | #C66747 | CTA highlights, hover states, seasonal banners |
| Soft Cream | #D7C1A9 | Background panels, newsletter, product detail blocks |
| Olive Brown | #6C5140 | Editorial banners, outdoor/lifestyle sections |
| Black | #000000 | Text, nav, buttons |
| White | #FFFFFF | Main background, product grids |

## Colour usage rule

Use black and white for the main store UI. Use the brand colours to create mood.

Good balance:

```txt
70% white / black ecommerce layout
20% product and lifestyle photography
10% brand colours
```

Do not place all four brand colours in one section. Pick one brand colour per block.

## Image style

Use:
- Clean studio shots
- Lifestyle movement shots
- Close-up fabric shots
- Product-on-model shots
- Outdoor active shots

Avoid:
- Mixed lighting
- Busy backgrounds on product cards
- Low-resolution screenshots
- Product cards with different crop sizes

## Component design rules

- Use consistent spacing.
- Keep cards simple.
- Use short headings.
- Use large images.
- Use the same button styles across the site.
- Keep product cards clear.
- Use one CTA per section where possible.

## Button styles

### Primary button

```txt
Black background
White text
Used for Add to cart, Shop now, Checkout
```

### Secondary button

```txt
White background
Black border
Used for View collection, Learn more
```

### Accent button

```txt
Burnt clay background
White text
Used for campaign highlights
```

## Mobile layout rules

- Put products near the top.
- Use horizontal scroll for product rows.
- Use 2-column product grids.
- Use a bottom sticky Add to Cart on product pages.
- Use filter drawer instead of sidebar.
- Keep hero copy short.
- Avoid popups at first load.
